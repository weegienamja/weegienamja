I work on networking automation and internal tooling at Cisco by day, and on side projects with no business case and no deadline by night. CCNA-certified, finishing a BSc at Glasgow Caledonian, fluent in being mildly annoyed by manual processes.

The thing I find most interesting right now is the seam between networking and AI agents. I use LLMs as an execution layer over real infrastructure, not as a chatbot bolted onto a dashboard. Most of my recent work is some variation of that.

### Currently building

- **[WindForge](https://github.com/weegienamja/WindForge)** — wind turbine site screening engine. TypeScript monorepo, ~680 tests, scoped npm packages.
- **[Anime-AI-Upscaler](https://github.com/weegienamja/Anime-AI-Upscaler)** — Electron + React app for AI upscaling and frame interpolation.
- **vManage MCP Server** *(Cisco)* — 20-tool MCP server over Cisco SD-WAN telemetry, with a correlation engine. Pydantic-validated, DevNet-tested.
- **GlobeScraper** *(private)* — Cambodia relocation and rental data platform. Next.js, Prisma, Supabase.

### Stack

Python · TypeScript · Next.js · Pydantic · Prisma · Postgres · Cisco kit. Strict types and `Result<T, E>` over thrown exceptions, mostly because I don't trust past-me.
